# Version Snapshots

This directory stores immutable snapshots of released versions.

## Rule

- Do not edit files inside an existing version folder after release.

## Structure

- versions/vX.Y.Z/index.html
- versions/vX.Y.Z/main.js
- versions/vX.Y.Z/config.js
- versions/vX.Y.Z/styles.css
- versions/vX.Y.Z/glossary.html
- versions/vX.Y.Z/data/
